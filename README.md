# JavaFeatureSignature

## Setup

1. Clone the repository.
2. Compile the Java code.
3. Write the command to run the `Main.java` file on your local machine in the `analysis.py` script.
4. Create a virtual environment and install the dependencies in `requirements.txt`.
5. Activate your virtual environment.
6. Run `repos.py` to gather the list of repositories.
7. Run `analysis.py` to perform the analysis.
8. Run `plots.py` to generate visualizations of the data.
